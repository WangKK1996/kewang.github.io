VPS + docker快速上手hadoop(linux)

发表于 2018-12-03 | 分类于 日常瞎玩 ， CS之路 |
| 字数统计: 543 | 阅读时长 ≈ 2

本篇不是教程！不是教程！不是教程！算是一个资源汇总，一小时内搞定hadoop集群，上手hive命令不是梦。

前提是：

  1. 装好了docker，docker安装文档
  2. 装好了docker-compose（1.10.0版本及以上），docker-compose的官方安装文档
  3. 拥有git账号；
  4. 拥有docker hub账号；
  5. 一个至少2G内存的linux虚拟机/服务器。（内存不够的话，跑hive可能会被强制kill掉）

废话与背景介绍

听闻hadoop环境配置令人作呕，这时候用docker起hadoop，作为练手就非常合适了。

上手只是玩一玩，熟悉一下hadoop架构，如果真的有大量数据处理需求的话，还是要老老实实自己搭集群哦。
